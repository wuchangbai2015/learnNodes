1   简介

一种新的AVR仿真软件。可以在任何支持avr-gcc的平台上运行，使用avr-gcc对寄存器的定义创建新的AVR设备。其内核易于定制，现在对于<=128KB Flash的设备支持的很好。

通过载入ELF文件来调试。

参考文献：

  1. [r] simavr： https://gitorious.org/simavr
  2. [r] simavr GetStarted： https://gitorious.org/simavr/pages/GetStarted
  3. [u] Debugging AVR code in Linux with simavr： http://www.instructables.com/id/Debugging-AVR-code-in-Linux-with-simavr/?lang=zh

2   simavr简介

支持的IO外设：

  1. EEPROM
  2. IO端口，包括中断
  3. 定时器8和16bit，支持一般、CTC、FastPWM、溢出中断
  4. UART，包括TX和RX中断
  5. SPI主从，包括中断
  6. I2C主从
  7. 外部中断，INT0等
  8. ADC
  9. 自编程，如bootloader

可以模拟的内核，也易于继续添加：

  1. AT90USB162，包括USB
  2. ATmega1280
  3. ATmega128
  4. ATmega16M1
  5. ATmega164/324/644
  6. ATmega48/88/168/328
  7. ATmega8
  8. ATtiny25/45/85
  9. ATtiny44/84
  10. ATtiny2313
  11. ATtiny13

其他支持：

  1. gdb完全可用(gdb服务器)，包括很酷的"passive mode"
  2. 可以简单的支持VCD(Value Change Dump)，并展示为可视化的波形图
  3. 很多实际例子，包括OpenGL渲染的显示
  4. 支持Arduino，但不含IDE

2.1   安装

依赖：

  1. gcc
  2. make
  3. gcc-avr
  4. avr-libc
  5. libelf
  6. freeglut

下载代码： http://gitorious.org/simavr/simavr/archive-tarball/master 。

解压后，安装:

cd simavr-simavr
make all

测试安装是否正常:

make -C tests run_tests

2.2   入门

运行:

./simavr/run_avr firmware.afx
./simavr/run_avr -m atmega88 -f 8000000 firmware.hex
./simavr/run_avr ./tests/atmega88_example.afx

还可以构造虚拟PCB，去例子目录看看。 avr_irq_register_notify() 可用于注册回调函数。
