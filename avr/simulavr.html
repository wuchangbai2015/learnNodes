2   Simulavr简介

这个项目处于过度时期，现在已经开始了向C++移植的Simulavrxx项目了。不过Simulavrxx还在开发中，而且与Simulavr有明显不同，现在已经是两个独立的项目了。

3   simulavr的基本使用

编译安装SimulAVR

$ ./configure --with-avr-includes=/usr/local/AVR/avr/include --prefix=/usr/local/AVR
$ make
# make install

编译安装AVR-GDB (6.x的版本源码):

$ ./configure --target=avr --prefix=/usr/local/AVR
$ make
# make install

编译选项 ：需要在编译时提供调试符号 avr-gcc 使用-Wa 和-gstabs 。

启动模拟器服务器

$ simulavr --device atmega8 --gdbserver

然后模拟器就已经准备好了，等待客户端连接到TCP:1212。

然后的avr-ddd调用avr-gdb指令(我的Debian上没有这个玩意):

$ avr-ddd --debugger $AVRROOT/bin/avr-gdb
(gdb) file stdiodemo/stdiodemo.elf
(gdb) target remote localhost:1212
(gdb) load
(gdb) break stdiodemo.c:53
(gdb) step
(gdb) continue

剩下的事情就是看寄存器。

有个工具叫 AVR Insight(WinAVR)，也可以用于连接提模拟器服务器。
