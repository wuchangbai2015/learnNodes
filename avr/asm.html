2   Hello World

2.1   第一个例子ATtiny13

.equ    DDRB,   0x17
.equ    PORTB,  0x18

.org    0x00

reset:          ;从0x00开始是中断向量表
    rjmp    main
    rjmp    default_interrupt
    rjmp    default_interrupt
    rjmp    default_interrupt
    rjmp    default_interrupt
    rjmp    default_interrupt
    rjmp    default_interrupt
    rjmp    default_interrupt
    rjmp    default_interrupt
    rjmp    default_interrupt

default_interrupt:  ;缺省中断向量
    reti

main:               ;主函数
    sbi     DDRB,   4
    sbi     PORTB,  4

end:                ;末尾维持当前状态
    rjmp    end

最开始的两行，定义了DDRB和PORTB，对应他们的寄存器地址。具体可以去 /usr/lib/avr/include/avr 里面找对应文件，如 iotn13.h 。

.org 0x00 用于定位代码所在位置。

reset 标签所在位置开始定义中断向量表。对于ATtiny13来说，中断向量表共10项：

  1. 0x00：RESET
  2. 0x01：INT0
  3. 0x02：PCINT0
  4. 0x03：TIM0_OVF
  5. 0x04：EE_RDY
  6. 0x05：ANA_COMP
  7. 0x06：TIM0_COMPA
  8. 0x07：TIM0_COMPB
  9. 0x08：WDT timeout
  10. 0x09：ADC complete

具体也是可以去 iotn13.h 中找到的。然后定义个缺省的中断处理器，直接返回即可。也有人将其定义到0x00，即找不到该中断就去复位。

主函数则很简单，就两行。 sbi DDRB, 4 是将DDRB寄存器的bit4置位。这样PB4就变为输出了。 sbi PORTB, 4 则是将PB4置为高电平，我这里是接了个LED。

最后可选的 end 标签，进行死循环。这里我实际测试可有可无，也许对于需要持续运行的程序是很必要的吧。

这个程序编译出来共28字节。去掉 end 则为26字节。与此功能相同的C程序则是74字节。里面可见根本没有使用 sbi 和 cbi 指令。可见为了兼容性，还是牺牲了不少东西的。

我对这个大小的输出还是比较满意的。接下来就是研究下如何闪耀。每个指令都是2字节的，即16bit，估计一部分指令是内置参数的，即立即数。

编译的步骤:

avr-as -mmcu=attiny13 -o al.o al.s
avr-ld -o al.elf al.o
avr-objcopy -j .text -j .data -O ihex al.elf al.hex

2.2   延时的例子分析

一个16bit的 delay() 函数，反汇编的结果为8个指令，居然比8bit的版本还段了14个指令。

C代码:

void delay(uint16_t millis) {
    uint16_t i,k;
    for (i=0;i<=millis;i++) {
        k++;
    }
}

其反汇编代码为:

delay:
    ldi     r18, 0x00
    ldi     r19, 0x00
    subi    r18, 0xff
    sbci    r19, 0xff
    cp      r24, r18
    cpc     r25, r19
    brcc    delay+0x4
    ret

调用端的代码，参数为255:

main:
    ; ... ...
    ldi     r24, 0xff
    ldi     r25, 0x00
    rcall   delay

逐步的分析：

  1. ldi r18, 0x00 是将立即数载入寄存器
  2. subi r18, 0xff 是常量减法
  3. sbci r19, 0xff 是常量带进位减法
  4. cp r24, r18 是直接比较两个寄存器
  5. cpc r25, r19 是带进位比较两个寄存器
  6. brcc delay+0x4 是当进位清除时跳转

对于16bit的实现，r24和r18对应低字节，r25和r19对应高字节。
